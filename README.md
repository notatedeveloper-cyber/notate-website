# Notate Website

Marketing / launch landing page for **Notate** — pinned visual feedback on any page, exported to Linear/Jira.

The site is a single self-contained file: [`index.html`](index.html) (all CSS/JS inline).

## Run locally

Open `index.html` directly in a browser, or serve it:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
